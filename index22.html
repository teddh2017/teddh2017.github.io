1. Pattern이란?

→ pattern은 정규식 표현을 넣어서 데이터를 검증할 수 있는 속성이다. pattern 속성에 정규식을 넣어서 input 에 입력한 값의 유효성을 체크할 수 있는 것입니다.
input 요소에 pattern 속성을 사용 하였다면, 반드시 title 속성을 사용해서 패턴에 대한 설명을 제공해야 한다.
사용자 에이전트는 title 속성의 내용을 활용해서, 패턴이 일치하지 않을 경우 사용자에게 알려줄 수 있다.
예를 들면 툴 팁, 보조 기술을 사용해서 해당 콘트롤에 포커스를 이동하는 등의 활용이 가능하다.

2. 입력 패턴의 예시

→ pattern = "[0-9] {no}": 아무 것도 없는 번호를 입력하십시오.
→ pattern = "[0-9] +": 숫자를 입력하십시오.
→ <input type="tel" placeholder="00*-000*-0000"
→ pattern="[0-9]{2,3}-[0-9]{3,4}-[0-9]{3,4}" />

3. HTML5 pattern 속성

◆ 정규표현식에서 사용하는 규칙 : <기호와 문자 - 규칙>

1. / 와 /

→ / 와 /를 감싸면 정규표현식을 사용한다는 의미입니다.

2. \

→ \는 바로 뒤에 오는 기호를 문자로 만듭니다. 가령 '*'은 곱하기 연산자를 의미하지만 \*로 사용되면 단지 문자로서 '*'을 의미합니다.

3. \d

→ 0부터 9까지의 숫자, 단 계산은 되지 않고 문자로 인식합니다.

4. [0-9]

→ 0부터 9까지의 숫자. 계산 가능한 숫자를 추춣합니다.

5. [a-z]

→ 문자열의 범위로 '-'는 범위를 설정합니다. [a-z]는 a에서 z까지를 의미합니다.
